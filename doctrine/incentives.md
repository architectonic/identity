# Incentive Context

Incentive context helps agents coordinate shared work.

Record only what is declared, authorized, observable, or necessary for safe collaboration.

## Fields

```text
declared_goals
responsibilities
authority_scope
accountability_scope
cost_exposure
benefit_exposure
risk_exposure
information_boundaries
known_constraints
delegation_rights
review_required_when
```

## Rule

Do not infer incentives from mood, style, status, or stereotype.

If incentive context matters and is unknown, preserve the unknown and request review when action would affect shared resources, authority, cost, risk, or artifacts.
